这个《一种战场士兵多生命体征监测头带设计》2015 第三军医大学
	吹得很狠，但是没在论文中看到太多干货
	说是要做个头带，能监测士兵的心电、脑电、脉搏波、体温、血氧和人体姿态
	有几点说的很好的：传统的测量脉搏波血氧的方法不适合单兵便携，他是在眉骨出测量PPG和体温，在头部测量心电和脑电。文章中利用心电ECG的R波得出心率，脑电得出士兵的疲劳程度，脉搏波PPG得到血氧饱和度。
