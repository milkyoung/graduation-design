	这篇文章很新，然后总结了心率监测中的4种噪声:基线漂移（呼吸）、工频干扰（交流电源）、肌电噪声、运动伪差（人的运动）
	她的做法是在手腕处测量得到PPG，但是她没有自己做硬件，是用的别人文章的数据，所以她就是提出了一个算法

这个《徐海津. 抗运动伪影下基于PPG的心率估计方法研究与应用[D].电子科技大学,2018.》
	这篇文章就是专门做运动情况下的心率监测
	他最后的结果是做了一套智能手环，但是集成度不行，太大了太杂了
	再就是采集的量太多了，包括手腕上的PPG、手腕上的加速度、三路ECG心电信号(这个是用来测量真实的心率的)
	晚上还可以继续看一看深挖一下这篇文章
	绪论部分对于之前的东西总结的挺好的，总结了基于PPG信号的心率监测的过程：
		1.时域下运动伪迹去除
		2.时频转换
		3.心率谱峰跟踪：机器学习方法就在这里用上了
	使用的去噪算法：
		使用了新型的处理非平稳信号的变分模态分解（VMD）：把PPG和加速度信号分解成很多模态，去除高频和低频，再去掉PPG中与加速度相近的模态，就ok了
		希尔伯特变换（HHT）
	他的去噪的步骤如下：
		1.基于HHT的心率估计初始值优化
			首先将信号进行经验模态分解（EMD），得到很多歌频率不同的本征模态函数，高配版小波变换，不依赖基函数，是一种自适应滤波算法·
			说白了就是通过HHT代替傅里叶转换也是把时域信号转化到频域，然后在这个频域里找到静止状态下的心率（注意此时是默认没有运动的）
		2.基于VMD算法的运动能够PPG信号去噪
			VMD是EMD的一种改进
			具体就是:
				1.对PPG信号进行VMD分解，然后通过分解结果把高频的波去掉
				2.对于第一步中剩下的那些低频的模态，遍历各个加速度通道中的能量大于50%的频率，如果很不幸第一步剩下的这些中包括加速度的频率，就把剩下的这些中的去掉
				3.最后就用这些剩下的模态来重构PPG信号就可以了
		3.时频转换
			FFT
		4.谱峰追踪
			根据上一个时间窗口得到的心率数据来寻找当前窗口的心率谱峰，所以第一步的那个初始值就是给第一个窗口用的，因为第一个窗口没有上一个窗口的值嘛
			具体如下：
				1.设定一个时间窗，时间窗的长度为6s，步进时间为2s
				2.根据上一个时间窗的波峰位置为缩小这一次的寻找范围，于是就找到了几个不错的波峰
				3.然后确认机制就是看这些波峰是否存在对应的谐波，存在就稳得是心率对应的波峰了

PPG去噪算法：
	小波变换：在B站上看了一个视频
	独立成分分析：假设心率信号和干扰信号是独立的，然后就可以分离出来了
	自适应滤波器：能够自适应优化参数的卡尔曼滤波器。这里就有很多研究者用加速度信号作为参考信号了
		在B站上看了卡尔曼滤波的视频，就是对一个数据进行两个数据源的融合，一个是理论上的数据比如公式推出来的，一个是实际测出来的，然后这两个数据加权相加就得到了最后的结果

=====================================应用背景、需求相关======================================

军事头戴式测量：
	传统的体征测量方法和测量方式不适合用于便携式体征监测装置，如，仍采用粘贴电极容易产生皮肤过敏、瘙痒等；从胸部测量心电信号和手指测量脉搏波信号，对士兵活动正常活动带来不便；
	传统的血氧检测位于手指、手腕或耳垂,易受人体活动而影响检测精度。
	反正就是说你在手腕手指处测量的话不得行，因为会影响人的正常活动或者这些地方动作太大了容易干扰
