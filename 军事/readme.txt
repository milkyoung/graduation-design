这个《一种战场士兵多生命体征监测头带设计》2015 第三军医大学
	吹得很狠，但是没在论文中看到太多干货
	说是要做个头带，能监测士兵的心电、脑电、脉搏波、体温、血氧和人体姿态
	有几点说的很好的：传统的测量脉搏波血氧的方法不适合单兵便携，他是在眉骨出测量PPG和体温，在头部测量心电和脑电。文章中利用心电ECG的R波得出心率，脑电得出士兵的疲劳程度，脉搏波PPG得到血氧饱和度。
	但是有个问题，他最后实验结果没提到最后结果的准确性，那就是不准确了咯，可以

这个《陈建锋. 头戴式生命体征参数采集装置的研究与实现[D]. 重庆邮电大学, 2016.》
	这个要做这方面的话可以在他的基础上改进着做，跟我的想法还挺像的
	头带式集成，根据心电的R波测量出心率，从PPG的PWTT测量得到血压，，从PPG信号直接得到血氧饱和度，然后把这些数据通过蓝牙传输到身上携带的北斗定位装置上，再转发给总部。PPG信号的采集在额头部位，心电信号的采集在脸颊部位
	最后结果说是：心率测量误差在4%以内，血氧饱和度在3%以内，血压的话收缩压在5mmHg以内舒张压在7mmHg以内，装置的功耗为205mW
	文章自己总结了几个不足的地方：
		1.去除运动伪迹干扰：现在的智能在静止的状态下正常的检测，运动状态下，心电和PPG信号将产生陡变和肌电干扰，有较大的误差
		2.提高抗电磁干扰能力，这个就不算我的事了
		3.提高舒适度
		4.降低功耗
