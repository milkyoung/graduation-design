怎么说呢，老师给提了温差发电这个方向，具体做的事情还是挺有意义的，感觉有实际意义并且相对而言也不是那么难以实现。但是我个人而言对这个不是那么感兴趣，但是鉴于这个方向很靠谱我就先看看师兄的论文找找感觉吧

看了师兄的毕设论文，总结一下把
	整体做的就是一个可温差发电的水壶，就是用火去加热水壶的时候能用火和被加热的额水的温差来发电，然后存在类似于电池的超级电容器里，可以拿来用
	工作：
		结合两种MPPT控制方法提出了一种控制温差发电工作在最佳状态的方法，然后simulink仿真
		针对超级电容器设计了电压均衡方案
		然后把这些东西都按在水壶上进行了实验

	具体做的就是用暖气片上面的温差发电片发电，用这个电来驱动一个电机，电机转动帮助这个暖气片的散热
